# Fleetwood Mac Fan Page

A portfolio project showcasing a fan page for **Fleetwood Mac** with band information, albums, and live performance videos. Built with **HTML, CSS, and Node.js** with a SQL backend (for albums or other dynamic content).

---

## Features

- Home page with hero image and brief introduction.
- About page with band history and member profiles.
- Albums page with album artwork, Spotify embeds, and a sorting option.
- Live page with embedded YouTube videos of performances.
- Responsive design for desktop and mobile.
- Stylish navigation and consistent layout across pages.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/en/) installed.
- SQL database (MySQL, SQLite, or PostgreSQL) if using dynamic album data.

### Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/my-first-project.git
cd my-first-project
```

2. Install dependencies:

```bash
npm install
```

3. Set up your database and configure connection in your Node.js project.

4. Start the server:

```bash
npm start
```

5. Open your browser and go to:
```bash
http://localhost:3000
```
